A pseudo-3D arcade racing game modeled after the classic game "Pole Position". Steer your car along the twisting track and accelerate to achieve the highest score! Supports keyboard and touch controls.

## Features (obstacles branch)
- 20 traffic cars to dodge, each with a random color, lane, and speed
- Colliding cuts your speed dramatically and flashes a CRASH! overlay
- 1.5-second invincibility window after each crash
- CRASHES counter on the HUD
