# Gabby Dot Com

A stylized, interactive webpage featuring a mischievous, adventure-loving cat named Gabby. Set against a starry night sky, Gabby wanders the screen following your cursor or touch.

Beware! If you click near the navigation bar, there's a 50% chance Gabby will playfully knock it down, followed by a cute victory spin!
