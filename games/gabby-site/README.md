# Gabby Site

A cute and professional personal website featuring a "punkish/pastel" pink and black aesthetic with twinkling background effects. Features standard tabs (Home, About, Contact), pig latin text, an image placeholder, and an interactive, mischievous virtual cat that roams the screen and sometimes knocks down website elements!
