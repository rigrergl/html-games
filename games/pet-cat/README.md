# Pet Cat

A relaxing, pixel-art style game where you adopt and care for a virtual cat. Brush them to hear them purr, give them treats, and build your relationship over time to unlock new toys like a dangly string and a laser pointer.

## Features
* Multiple cat types to choose from (Orange, Black, White, Tuxedo).
* Multiple save slots via `localStorage` so you can care for a whole shelter.
* Synthesized purring audio (Web Audio API) when you brush them.
* Relationship progression system unlocking new interactive toys.
* 100% Vanilla HTML, CSS, and JS. Single-file build.
